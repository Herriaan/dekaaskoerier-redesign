# De Kaaskoerier — redesign mockup

Klikbaar ontwerpvoorstel voor [dekaaskoerier.nl](https://dekaaskoerier.nl), gemaakt door **Katama** (juni 2026).

Statische HTML/CSS-mockup, geen webshop. Concept **C — "Smaak op de markt"**: merkbeleving bovenin, een vind-de-kraam-marktkalender als ruggengraat, en een schaalbare lokale-SEO-laag (markt = pagina) als fundament.

## Pagina's
- `index.html` — homepage
- `vind-je-kraam.html` — marktkalender + filterbare kaart + lijst
- `markt-alkmaar.html` — markt-detailtemplate (lokale SEO + LocalBusiness-schema)
- `koerier-matteo.html` — koerier-profiel (De Noord-Hollandse) + assortiment

## Let op
Dit is een ontwerpvoorstel. Marktdagen, tijden en reviews zijn deels voorbeeld-data. Foto's zijn stock (Pexels-licentie, zie `assets/img/credits.md`).
